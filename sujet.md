# Sujets

Chaque étudiant doit versionner dans sa branche git de la forme
`prenom-nom/sujet-en-4-ou-5-mots` toute documentation, schéma, pptx, markdown et
le code source de la démo.

## Use cases avec Open WebUI

- Présenter Open WebUI : installation, architecture et choix des modèles.
- Démontrer le maximum de cas d'usage possibles : génération, résumé, Q&A, RAG
  intégré.
- Montrer les fonctionnalités avancées : gestion des conversations, plugins,
  personnalisation. Donner des exemples pertinents.
- Comparer Open WebUI avec d'autres solutions et discuter des limites.
